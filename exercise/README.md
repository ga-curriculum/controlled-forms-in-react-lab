# ![Controlled Forms in React Lab - Exercise](./assets/hero.png)

All in one component, students should implement a controlled form that adds an object to a list in state- that list should be prepopulated with one or two example objects in state, and should have a list displaying each member of the array.

Basically, an all-in-one to-do list, but we're trying to be more creative than a to do list. Hopefully we come up with a clever app that manages a list and adds new items to it from a form, but we've done that so many times now, it's tough to keep making original ideas! Any app that does that sort of thing should be totally fine.
